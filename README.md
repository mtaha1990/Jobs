<!-- GETTING STARTED -->
## Getting Started



### Prerequisites


* npm
* node

### Installation



1. Clone the repo
   ```sh
   git clone https://github.com/mtaha1990/Jobs.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```


<!-- USAGE EXAMPLES -->
## Usage

1. run server.js
   ```sh
   node backend/server.js
   ```
2. browse the index page on browser page in frontend/index.html
